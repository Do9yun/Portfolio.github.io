엣지와 영역

에지 검출의 기본 개념

에지는 영상에서 밝기(명암)가 급격히 변하는 부분으로, 주로 물체의 윤곽이나 경계를 나타냅니다. 이런 에지를 검출하면 영상에서 중요한 정보를 추출할 수 있습니다. 예를 들어, 물체의 형태를 파악하거나 객체를 분리하는 데 사용됩니다. 에지 검출은 컴퓨터 비전에서 가장 기본적이고 중요한 작업 중 하나로, 객체 탐지, 영상 분할, 패턴 인식 등 다양한 응용 분야에서 활용됩니다.

미분의 정의

미분은 변수 x가 아주 작은 값 δx만큼 증가할 때 함수 f(x)의 변화율을 측정하는 수학적 기법입니다. 이는 다음과 같이 수식으로 표현됩니다:

디지털 영상에서 미분

디지털 영상에서는 연속적인 함수 대신 화소(pixel) 값으로 이루어진 이산적인 데이터를 다루기 때문에, 미분을 아래와 같이 간소화하여 표현합니다

여기서 δx = 1로 가정합니다. 이는 인접한 화소 간의 밝기 차이를 계산하는 방식입니다.

디지털 영상에서 미분의 구현

디지털 영상에서 미분은 필터 u를 사용해 계산합니다.

  • 필터는 컨볼루션(convolution) 연산을 통해 원본 영상의 밝기 차이를 계산하며, 아래와 같은 필터를 사용합니다

u = [-1, 1]

  • 이 필터를 원본 영상에 적용하면, 각 화소에서 오른쪽 인접 화소와의 밝기 차이를 계산합니다.

연속 함수와 디지털 영상에서의 미분

연속 함수의 미분
함수 f(x)가 점 x에서 x + δx로 변화할 때의 기울기를 구합니다. 기울기는 δx가 작을수록 정확히 측정됩니다.

디지털 영상의 미분

  • 원본 영상 f: 입력 영상에서 각 화소는 특정한 밝기 값을 가집니다.
  • 필터 u = [-1, 1]: 이 필터를 원본 영상에 적용하여 인접 화소 간의 차이를 계산합니다.
  • 결과 영상 f': 필터를 적용한 결과로 에지(명암 변화가 큰 부분)만 강조된 새로운 영상이 생성됩니다.
명암 변화가 없는 부분에서는 0, 변화가 있는 부분에서는 차이 값을 반영합니다.
예: 결과값: [0, 0, 0, 3, 0, 0, 0].
