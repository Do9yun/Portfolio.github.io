엣지와 영역

엣지 검출:

엣지(edge)는 이미지에서 밝기 값이 급격히 변화하는 영역을 나타냅니다. 엣지는 객체의 경계선을 감지하는 데 사용되며, 대표적인 엣지 검출 알고리즘은 다음과 같습니다.

  • Sobel: 수평 및 수직 방향으로 엣지를 검출합니다.
  • Canny: 노이즈 제거 후 강도와 방향을 기반으로 엣지를 검출합니다.

영역 분석:

영역(region)은 서로 연결된 픽셀 집합으로 정의됩니다. 일반적으로 특정 색상 또는 밝기 값을 기준으로 영역을 분리합니다. 주요 영역 분석 기법은 다음과 같습니다.

  • Flood Fill: 주어진 점에서 시작하여 인접한 픽셀들을 채웁니다.
  • Contour Detection: 객체의 외곽선을 추출합니다.

주요 코드:

edges = cv.Canny(img, 100, 200)
Canny 엣지 검출 알고리즘을 사용하여 엣지를 검출합니다.

  • plt.imshow(edges, cmap='gray'): 검출된 엣지를 시각화합니다.

영역 분석 코드:

contours, _ = cv.findContours(bin_img, cv.RETR_EXTERNAL, cv.CHAIN_APPROX_SIMPLE)
이진화된 이미지에서 외곽선을 감지합니다.

  • cv.drawContours(img, contours, -1, (0, 255, 0), 2): 검출된 외곽선을 원본 이미지 위에 그립니다.

영상의 미분:

미분은 영상에서 밝기 값의 변화율을 측정하는 과정으로, 엣지 검출에 핵심적인 역할을 합니다. 디지털 영상에서 미분은 이산적인 데이터를 다루기 때문에 아래와 같은 방식으로 계산됩니다.

  1. 미분의 정의:

    미분은 변수 x가 아주 작은 값 δx만큼 증가할 때 함수 f(x)의 변화율을 측정하는 수학적 기법입니다. 아래는 미분의 수학적 정의입니다:

    f'(x) = lim(δx→0) [(f(x+δx) - f(x)) / δx]

  2. 디지털 영상에서 미분:

    디지털 영상에서는 연속적인 함수 대신 픽셀 값으로 이루어진 이산적인 데이터를 다루기 때문에 미분은 아래와 같이 간소화하여 표현됩니다:

    f'(x) ≈ f(x+1) - f(x)

  3. 디지털 영상에서 미분의 구현:

    디지털 영상에서 미분은 필터를 통해 계산됩니다. 예를 들어, 아래의 필터를 사용하여 미분을 구현할 수 있습니다:
