OpenCV

  1. import cv2 as cv

    OpenCV 라이브러리를 cv라는 이름으로 가져옵니다. 이 라이브러리는 컴퓨터 비전 작업에 사용됩니다.

  2. import sys

    Python의 시스템 관련 기능을 사용할 수 있도록 sys 모듈을 가져옵니다.

  3. img=cv.imread('soccer.jpg')

    soccer.jpg라는 이미지 파일을 읽어 img 변수에 저장합니다. 이미지 파일은 스크립트와 동일한 디렉토리에 있어야 합니다.

  4. if img is None:

    파일이 존재하지 않거나 읽기에 실패한 경우, img 변수는 None 값을 가집니다. 이를 확인하는 조건문입니다.

  5. sys.exit('파일을 찾을 수 없습니다.')

    이미지 파일을 찾지 못했을 경우, 프로그램을 종료하고 '파일을 찾을 수 없습니다.'라는 메시지를 출력합니다.

  6. cv.imshow('Image Display', img)

    읽어들인 이미지를 새 창에 표시합니다. 창의 제목은 'Image Display'입니다.

  7. cv.waitKey()

    키 입력이 있을 때까지 프로그램을 대기 상태로 유지합니다. 이 함수는 창이 바로 닫히는 것을 방지합니다.

  8. cv.destroyAllWindows()

    모든 OpenCV 창을 닫습니다.
