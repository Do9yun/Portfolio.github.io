Deep Learning

딥러닝의 기초 이론

딥러닝 기초, 다양한 학습 모델, 활성화 함수, 손실 함수, 딥러닝 학습 과정, 과적합 문제와 해결 방안, 최적화 알고리즘, 그리고 딥러닝의 이점 설명.

파이토치 개요 및 실습

PyTorch를 활용해 딥러닝 기초를 실습하는 강의 자료로, 예측 문제를 통해 신경망을 설계하고 GPU를 활용한 효율적인 학습 방법을 다룹니다. 텐서의 기본 개념과 PyTorch 모듈(torch.nn)을 사용한 신경망 구성, 데이터 분리, 손실 함수와 역전파를 통한 학습, 그리고 결정계수(R²)를 활용한 모델 평가 방법을 포함하며, GPU를 활용한 병렬 연산으로 학습 성능을 최적화하는 과정을 설명합니다.

딥러닝 구현

MINIST 데이터셋을 활용하여 숫자를 분류하는 딥러닝 모델을 PyTorch로 구현하는 과정에 대해 설명하며, 데이터 전처리, 텐서 변환, 정규화, 데이터 로더 설정, 모델 학습 및 평가의 전 과정을 다룹니다.

합성곱 신경망

합성곱 신경망(CNN)의 원리를 다루고, CNN의 작동 원리를 이해합니다.

합성곱 신경망과 튜닝 기법

CIFAR-10 데이터셋을 활용해 CNN 구조를 설계하고, 다양한 데이터 전처리 및 증강 방법(예: 이미지 뒤집기, 크기 조정, 밝기 조정 등)을 설명합니다. 또한, 드롭아웃과 배치 정규화를 포함한 과적합 방지 기술과 최적화 방법(Adam 등)을 다룹니다. 데이터 정규화 및 변환 기법과 CNN 구성, GPU 연동, 손실 함수 정의 등 실습 중심으로 구성되어 있습니다.

전이학습

사전 학습된 모델(ResNet-18)을 활용해 CIFAR-10 데이터셋을 학습하는 과정을 설명합니다.

순환신경망(RNN)

시간적 연속성이 있는 데이터를 처리하는 방법과 다양한 RNN 구조(다대다, 다대일 등)를 소개합니다. 또한, RNN의 한계와 이를 극복하기 위한 LSTM(Long Short-Term Memory)의 작동 원리를 설명합니다.

GAN

GAN의 원리와 구현, MNIST 데이터셋을 활용한 실습, 생성자와 판별자의 학습 함수화 및 반복문 최적화, 손실 함수 계산과 오차 역전파 과정을 설명합니다.

DCGAN

DCGAN의 구조와 원리를 중심으로, CNN과 역합성곱을 활용한 이미지 생성 과정, Convolution 연산의 수학적 계산, 생성자와 판별자의 손실 함수 설계, 그리고 SGD Momentum, RMSProp, Adam 등 최적화 알고리즘을 포함한 학습 방법에 대해 다루며, DCGAN을 활용한 데이터 전처리와 이미지 생성의 실제 구현 과정을 설명합니다.

PINN

PINN(Physics-Informed Neural Network)의 개념과 활용에 대해 설명하며, 자유낙하 데이터를 활용한 PINN 기초 실습, 그리고 상미분방정식의 정보의 유무에 따른 예측 정확도 대해 설명합니다.

PINN2

PINN(Physics-Informed Neural Network)을 이용한 감염병 모델링 및 예측에 대한 설명입니다. PINN의 원리를 이해하고 이를 통해 감염병 확산 방정식을 활용하여 데이터를 기반으로 감염자 수를 예측하는 신경망을 구현하는 내용을 다룹니다. SIR(Susceptible-Infectious-Recovered) 모델을 기반으로 상미분방정식 정보를 신경망 학습에 포함하여 데이터와 물리적 제약을 결합해 더 정확한 예측 결과를 얻는 방법을 설명합니다.
