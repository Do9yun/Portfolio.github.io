딥러닝의 기초 이론

딥러닝 기초, 다양한 학습 모델, 활성화 함수, 손실 함수, 딥러닝 학습 과정, 과적합 문제와 해결 방안, 최적화 알고리즘, 그리고 딥러닝의 이점 설명.

파이토치 개요 및 실습

PyTorch를 활용해 딥러닝 기초를 실습하는 강의 자료로, 예측 문제를 통해 신경망을 설계하고 GPU를 활용한 효율적인 학습 방법을 다룹니다. 텐서의 기본 개념과 PyTorch 모듈(torch.nn)을 사용한 신경망 구성, 데이터 분리, 손실 함수와 역전파를 통한 학습, 그리고 결정계수(R²)를 활용한 모델 평가 방법을 포함하며, GPU를 활용한 병렬 연산으로 학습 성능을 최적화하는 과정을 설명합니다.

딥러닝 구현

MINIST 데이터셋을 활용하여 숫자를 분류하는 딥러닝 모델을 PyTorch로 구현하는 과정에 대해 설명하며, 데이터 전처리, 텐서 변환, 정규화, 데이터 로더 설정, 모델 학습 및 평가의 전 과정을 다룹니다.

합성곱 신경망

합성곱 신경망(CNN)의 원리를 다루고, CNN의 작동 원리를 이해합니다.

합성곱 신경망과 튜닝 기법

CIFAR-10 데이터셋을 활용해 CNN 구조를 설계하고, 다양한 데이터 전처리 및 증강 방법(예: 이미지 뒤집기, 크기 조정, 밝기 조정 등)을 설명합니다. 또한, 드롭아웃과 배치 정규화를 포함한 과적합 방지 기술과 최적화 방법(Adam 등)을 다룹니다. 데이터 정규화 및 변환 기법과 CNN 구성, GPU 연동, 손실 함수 정의 등 실습 중심으로 구성되어 있습니다.
