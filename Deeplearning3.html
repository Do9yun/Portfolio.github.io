MINIST 데이터를 이용한 딥러닝 구현

1. MNIST 데이터 다운로드

PyTorch를 이용해 MNIST 데이터셋을 다운로드하고, GPU 또는 CPU를 자동으로 선택하도록 설정합니다.

2. 데이터 시각화

MNIST 데이터 중 하나의 이미지를 출력하며, cmap='gray_r'로 이미지를 흑백 반전으로 보여줍니다.

MNIST 데이터셋에서 처음 20개의 이미지를 2행 10열 그리드로 출력하며, cmap='gray_r'를 사용합니다.

3. 텐서 변환

MNIST 데이터를 텐서로 변환한 뒤, 데이터 타입, 크기, 최소값, 최대값을 확인합니다.

텐서로 변환된 이미지 데이터를 숫자로 표현된 형태로 확인합니다.

평균과 표준편차를 기준으로 정규화하고, 1차원 텐서로 변환한 후 크기와 최소/최대값을 확인합니다.

4. 데이터 분할과 DataLoader

학습용과 검증용 데이터로 나누고, DataLoader를 사용해 데이터를 로드합니다.

train_loader에서 몇 개의 배치로 데이터가 나뉘는지 확인하고, 첫 번째 배치의 크기를 출력합니다.

5. 신경망 정의

입력과 출력 노드 수를 설정하고, 은닉층이 1개인 신경망 모델을 정의한 후 내부 파라미터를 출력합니다.

입력층, 은닉층, 출력층과 ReLU 활성화 함수로 구성된 신경망 구조를 출력합니다.

6. 모델 분석 및 학습 진행

torchinfo 라이브러리를 사용해 신경망 모델의 개요를 출력하며, 각 층의 출력 크기와 파라미터 개수를 보여줍니다. 총 학습 가능한 파라미터 수는 101,770개입니다.

tqdm을 사용해 학습 데이터 로딩 진행 상황을 시각적으로 표시하며, n_train 변수에 전체 학습 데이터의 개수를 계산해 저장합니다.

난수 고정을 통해 결과 재현성을 보장하고, 학습률, 모델, 손실 함수, 최적화 함수를 설정한 후 평가 결과를 저장할 배열을 초기화합니다.

100번의 에포크 동안 학습을 수행하며, 각 배치에서 데이터를 GPU로 전송하고, 손실 함수 계산 및 역전파를 통해 파라미터를 업데이트하며 학습 정확도와 손실을 계산합니다.

테스트 데이터로 모델의 예측을 수행하고, 손실과 정확도를 계산한 후, 학습 및 테스트 결과를 기록하여 이후 평가를 위해 배열에 저장합니다.

학습의 초기 상태와 최종 상태에서의 손실과 정확도를 출력하여 모델 성능의 변화를 확인합니다.

학습과 테스트 데이터의 손실 곡선을 에포크별로 그래프로 출력하여 모델의 학습 상태와 일반화 성능을 시각적으로 보여줍니다.
그래프를 보면 학습 손실은 점차 감소하고 테스트 손실은 약간 증가하는 경향을 보입니다. 이는 모델이 과적합되고 있을 가능성이 있다는걸 보여줍니다.

그래프를 보면 학습과 테스트 데이터의 정확도 곡선을 에포크별로 그래프로 출력하여 모델의 학습 진행과 일반화 성능을 시각적으로 보여줍니다.
그래프를 보면 학습 정확도는 98%에 도달하고 테스트 정확도는 약 95%를 유지하는걸 볼 수 있습니다. 이는 모델이 전반적으로 높은 성능을 보이지만 테스트 데이터에 대한 일반화가 약간 부족할 수 있다는 의미로 해석할 수 있습니다.

테스트 데이터에서 첫 번째 배치를 가져와 모델의 예측 결과를 확인하고, 50개의 이미지를 "정답:예측" 형식으로 시각화하여 올바른 예측과 잘못된 예측을 구분해 출력합니다.
결과를 보면 대부분의 예측이 올바르지만 일부 오답이 존재하며 모델의 성능은 우수하나 특정 데이터 분포에 대해 오차를 줄이는 추가 개선이 필요할 수 있습니다.
