MINIST 데이터를 이용한 딥러닝 구현

1. MNIST 데이터 다운로드

PyTorch를 이용해 MNIST 데이터셋을 다운로드하고, GPU 또는 CPU를 자동으로 선택하도록 설정합니다.

2. 데이터 시각화

MNIST 데이터 중 하나의 이미지를 출력하며, cmap='gray_r'로 이미지를 흑백 반전으로 보여줍니다.

MNIST 데이터셋에서 처음 20개의 이미지를 2행 10열 그리드로 출력하며, cmap='gray_r'를 사용합니다.

3. 텐서 변환

MNIST 데이터를 텐서로 변환한 뒤, 데이터 타입, 크기, 최소값, 최대값을 확인합니다.

텐서로 변환된 이미지 데이터를 숫자로 표현된 형태로 확인합니다.

평균과 표준편차를 기준으로 정규화하고, 1차원 텐서로 변환한 후 크기와 최소/최대값을 확인합니다.

4. 데이터 분할과 DataLoader

학습용과 검증용 데이터로 나누고, DataLoader를 사용해 데이터를 로드합니다.

train_loader에서 몇 개의 배치로 데이터가 나뉘는지 확인하고, 첫 번째 배치의 크기를 출력합니다.

5. 신경망 정의

입력과 출력 노드 수를 설정하고, 은닉층이 1개인 신경망 모델을 정의한 후 내부 파라미터를 출력합니다.

입력층, 은닉층, 출력층과 ReLU 활성화 함수로 구성된 신경망 구조를 출력합니다.
